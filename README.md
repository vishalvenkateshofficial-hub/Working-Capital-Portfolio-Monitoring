# Working-Capital-Portfolio-Monitoring
End-to-end BA project — SQL Server + Power BI portfolio monitoring dashboard for BFSI working capital.

## What this is
Self-initiated, end-to-end BA lifecycle project for a working capital (OD/CC) portfolio 
monitoring solution — from AS-IS/GAP analysis through BRD/FRD, SQL data model, and a 
5-page Power BI dashboard. Built solo, requirements through delivery.
